Project Name
This project is a web application built with MongoDB, Spring Boot (Java), and React.

Description
The project is designed to showcase the integration of MongoDB, Spring Boot, and React to create a full-stack web application. It allows users to perform various operations on a data collection stored in MongoDB through a user-friendly interface built with React.


Technologies Used
MongoDB: create project connect to cluster using compass -->Download mongodb.compass   A NoSQL database used for storing and retrieving data.
Spring Boot: A Java framework used for building the backend API.
React: A JavaScript library used for building the user interface.

dependencies in Spring Initializr:Lombok, spring web, spring data mongodb,spring boot developer,

Installations:

Java Development Kit (JDK) versionm 17
Node.js the newest version
MongoDB the newest version


Installation

Backend setup:
Navigate to the backend directory: cd backend
Install dependencies: mvn install
Configure the MongoDB connection in application.properties
Run the backend server: mvn spring-boot:run




Frontend setup:
Navigate to the frontend directory: cd frontend
Install dependencies: npm install
Configure the API endpoint in src/api/index.js
Run the frontend server: npm start
Access the application in your browser at http://localhost:3000
